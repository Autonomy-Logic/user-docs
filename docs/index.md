# Autonomy Edge Documentation

Welcome to the Autonomy Edge user manual. This guide walks you through what the platform is, what each screen does, and how to put it all together to build, deploy, and share PLC programs from your browser.

If you're brand new, start with **[What is Autonomy Edge?](getting-started/what-is-autonomy-edge)** and then follow the **[Quick Start Guide](getting-started/quick-start)** to ship your first program in under fifteen minutes. If you're already on the platform and just want to look something up, jump straight to the section that matches the screen you're on.
