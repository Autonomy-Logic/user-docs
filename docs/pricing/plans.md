# Pricing & Plans

## Free During Beta

Autonomy Edge is currently in **beta**, and during this period the platform is **completely free** to use. No credit card is required, no hidden fees, and no feature limitations. You get full access to everything the platform has to offer.

> **Why free?** We believe the best way to build a great product is together with our users. During the beta, your feedback helps us shape the future of cloud PLC programming. In return, you get full access to all features at no cost.

---

## What's Included

During the beta, every user gets access to **all platform features** with no restrictions:

### Cloud PLC Editor
- Full browser-based **IEC 61131-3** programming environment
- Syntax highlighting, auto-complete, and error detection
- Support for **Structured Text (ST)**, **Ladder Diagram (LD)**, **Function Block Diagram (FBD)**, and **Instruction List (IL)**
- Custom programming with **Python** and **C++** function blocks

### Unlimited Projects
- Create as many projects as you need
- Organize with **folders** and **pinning** for quick access
- **Fork** existing projects to build on top of community work
- **Import and export** projects freely
- Set projects as **public** or **private**

### Orchestrators
- Create and manage **unlimited orchestrators**
- Group your devices and PLCs with powerful orchestration tools
- Full orchestrator overview with connected device status

### Device Management
- Connect **physical PLCs** or **virtual devices (vPLCs)**
- Monitor **CPU usage**, **memory**, **uptime**, and **network configuration** in real-time
- Create and manage vPLC devices directly from the platform
- View detailed device information including serial ports and network interfaces

### Cloud Deployment
- **One-click deployment** to connected devices from anywhere
- Compile and deploy your PLC programs directly from the browser
- Support for XML export format for external tools

### Live Debugging
- **Execute code** and monitor variables in real-time
- View execution logs and step through your programs
- Console and debugging tools integrated into the editor

### AI Assistant
- **AI-powered assistant** to help with IEC 61131-3 programming
- Get guidance on project setup, troubleshooting, and best practices
- Chat history to revisit previous conversations
- Available globally with a keyboard shortcut

### Documentation
- Comprehensive **built-in documentation** covering all platform features
- Full IEC 61131-3 language reference and examples
- Searchable with sidebar navigation

### Community Features
- **Public profiles** to showcase your projects
- Explore what other users are building
- **Activity history** to track your contributions

### Secure Authentication
- Sign in with **Google**, **Microsoft**, or **Apple**
- Enterprise-grade security with **JWT tokens**
- Email verification and secure password reset

### Continuous Updates
- Regular feature releases and improvements
- **Changelog** with detailed update notes
- Be the first to try new capabilities as they launch
