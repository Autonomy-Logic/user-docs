# Direct Messages and Group Chats

Direct messages (DMs) let you talk privately with other members, without posting to the public forum. You can also create group chats with several people.

**Where to find it:** Click **Messages** in the forum navigation, or go to `edge.autonomylogic.com/messages`.

> You must be **signed in** to send or receive messages.

---

## The Messages Page

The page has a two-pane layout:

- **Left sidebar**: The list of all your conversations.
- **Right pane**: The selected conversation, or an empty state if none is selected.

On mobile, the two panes stack vertically. You see the sidebar first, tap a conversation to see it.

---

## The Sidebar

### Header

- **Back to forum** arrow. Returns to the forum home.
- **Messages** heading.
- **✏️ New message** button. Starts a new 1:1 DM. Opens a panel to pick a recipient.
- **👥 Create group** button. Starts a new group chat. Opens a panel to set the group name and add members.

### Search box

Below the header, a **"Search conversations..."** field filters the list as you type. It searches conversation names and the last message content.

### Conversation list

Every conversation you've had appears as a row. For each row you'll see:

- **Avatar**: The other person's avatar (for 1:1 DMs) or the group avatar.
- **Name**: The other user's name, or the group name.
- **Last message snippet**: The beginning of the most recent message, truncated with `…`.
- **Timestamp**: Relative ("2h ago", "Yesterday").
- **Unread badge**: A small colored circle with a count of unread messages, if any.
- **Online dot**: A small green dot on the avatar corner if the person is currently online.

Click any row to open that conversation in the right pane.

### Group invitations section

If someone invited you to a group chat and you haven't responded yet, a separate section appears at the top of the sidebar showing pending invitations. Each invitation has:

- The group name.
- **Accept** button. Adds you to the group.
- **Decline** button. Rejects the invitation; the group creator is not notified beyond the default UI.

---

## Inside a Conversation (Right Pane)

### Header bar

- **Conversation name**: The other person's name, or the group name.
- **ℹ️ Info button** (group chats only): Opens a panel with group details.
- **Description**: If the group has one, it's shown in muted text.

### Message list

Messages are displayed chronologically, oldest at the top. The page auto-scrolls to the newest message when you open the conversation and when new messages arrive.

Each message shows:

- **Avatar** of the sender (left).
- **Name** and **timestamp**.
- **The message body**:
  - Plain text and rich text (links, **bold**, *italic*, etc.).
  - **Attachments** appear as download links or embedded previews.
  - **Images** appear as thumbnails you can click to view full-size.
  - **Emoji reactions** (if supported on your message) show below the body.

### Typing indicator

When the other person starts typing a reply, you see **"[name] is typing..."** below the latest message. It disappears when they stop or send.

### Message actions (on your own messages)

Hover over a message you sent. A **•••** menu appears with:

- **Edit**: Put the message into edit mode; save or cancel as usual.
- **Delete**: Removes the message from the conversation.

Note: once another person has already seen your message, deleting it is possible but the other person may have received an email or push notification about it before.

### The composer (bottom)

A simple text input at the bottom of the pane with:

- **😊 Emoji picker**: Opens a full emoji browser. Search by name, switch skin tone, see your recent emojis. Click an emoji to insert it.
- **📎 Attachment**: Opens your file picker. Max **10 MB per file**.
- **🖼️ Image**: Same as attachment but filtered to images. Renders inline in the message.
- **Drop zone**: Drop files or images anywhere on the composer to attach them.
- **Send button**: Sends the message. Disabled when the input is empty. Pressing Enter does not send; click the button or the conversation's send shortcut if configured.

## Starting a New DM

Click **✏️ New message** in the sidebar header. A panel opens with:

- A search input. Type a name or username to find someone.
- A list of matching members, each with avatar and name.
- Click a name → a new DM conversation opens in the right pane (or the existing one if you've already chatted).

> **Heads up:** If the other person has **"Allow group chat invitations"** turned off in their Privacy settings, you **can still start a 1:1 DM** with them. That toggle only affects group invitations.

---

## Creating a Group Chat

Click **👥 Create group** in the sidebar header. A panel opens with:

- **Group name**: A text field. Required.
- **Add members**: A search box. Type to find people; click their name to add them. Selected members appear as removable pills.
- **Create** button. Finalizes the group. The members are notified that you invited them (see "Group invitations section" above).

> **Tip:** Give groups clear names. "Modbus troubleshooting" is better than "chat 3". It's easier for invited people to decide whether to accept.

---

## Group Details

In a group chat, click the **ℹ️ Info** button in the header to open the group details panel. From here:

- View the full **member list**.
- Click **Add member** to search for and invite more people.
- Group owners can **remove members** from the list.
- Everyone can **Leave group**: that removes you from the conversation.

---

## Online Status

The small green dot on a user's avatar means they're currently online on Autonomy Edge. Absence of the dot doesn't guarantee they're away. It just means their session isn't actively connected at this moment.

---

## Unread Messages Reminder (Email)

If you have a direct message that's been **unread for more than 3 days**, the platform can send you a gentle reminder email. It's opt-in per user (on by default). See [Email and Privacy Preferences](email-preferences) for the toggle.

The reminder is rate-limited: you'll only get one email per 3 days at most, even if you accumulate many unread conversations.

---

## Emoji Picker Details

- Browse by category (smileys, people, food, activities, objects, etc.).
- **Search by name**: Type "cat" and you see every cat-related emoji.
- **Skin tone selector**: For emojis that support it.
- **Recent emojis**: Up to 2 rows of your most-used emojis at the top.
- **9 per row** in the grid.

## Quick Reference

| Action | Where |
|---|---|
| Start a 1:1 DM | Sidebar → ✏️ New message |
| Create a group | Sidebar → 👥 Create group |
| Accept/decline invitation | Top of sidebar (when present) |
| Attach a file | 📎 in the composer (or drag/drop) |
| Add emoji | 😊 in the composer |
| Edit your message | Hover message → ••• → Edit |
| Delete your message | Hover message → ••• → Delete |
| See group details | Header → ℹ️ |
| Leave a group | Group details → Leave group |
