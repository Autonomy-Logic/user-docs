# Platform Features

The Autonomy Edge platform provides a comprehensive suite of features for managing industrial automation projects in the cloud. This section covers the core functionality available through the web-based interface, including project management, orchestrator configuration, virtual PLC device management, and community collaboration tools.

## Overview

The platform is designed to streamline the development and deployment of IEC 61131-3 compliant PLC programs. Key features include:

- **Project Management**: Create, organize, and manage your automation projects with folder structures, visibility controls, and collaboration features
- **Orchestrator Management**: Configure and monitor edge orchestrators that coordinate your distributed automation systems
- **Virtual PLC Device Management**: Deploy and manage virtual PLC devices that run your automation programs
- **Community Features**: Discover and share projects with the Autonomy Edge community through activity feeds and search functionality

## Getting Started

To begin using the platform features, you'll need to have an active Autonomy Edge account. Once logged in, you can access all features through the main dashboard and navigation menu.

## Navigation

The platform uses a consistent navigation structure across all pages:

- **Header**: Contains the Autonomy Edge logo, account badge, notifications, and user profile menu
- **Sidebar**: Provides quick access to projects, orchestrators, and other key features
- **Main Content Area**: Displays the current page content and interactive elements
- **Right Sidebar**: Shows contextual information like organizations and recent changes
