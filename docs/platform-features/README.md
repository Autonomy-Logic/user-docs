# Platform Features

The Autonomy Edge platform gives you everything you need to create, deploy, and manage industrial automation projects from your browser. This section covers the core features: project management, orchestrator setup, and virtual PLC device management.

---

## What You Can Do

- **Project Management**: Create, organize, and manage your PLC projects with folders, visibility controls, and import/export.
- **Orchestrator Management**: Install and monitor orchestrators on your Linux machines to bridge the cloud and your hardware.
- **vPLC Device Management**: Create and manage virtual PLC devices that run your automation programs on your orchestrators.

---

## Getting Started

You need an Autonomy Edge account to access these features. Once logged in, everything is available from the main dashboard and navigation menu.
